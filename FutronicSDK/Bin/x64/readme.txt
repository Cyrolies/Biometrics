To run examples you should install "Microsoft Visual C++ Redistributable Package" and "Microsoft .NET Framework"

To run WorkedExNet8.exe, WorkedExVbNet8.exe install "Microsoft Visual C++ 2005 Service Pack 1 Redistributable Package MFC Security Update"
https://www.microsoft.com/en-us/download/details.aspx?id=26347

To run WorkedExNet10.exe, WorkedExVbNet10.exe install "Microsoft Visual C++ 2010 Redistributable Package (x64)"
http://www.microsoft.com/en-us/download/details.aspx?id=14632
and "Microsoft .NET Framework 4"
http://www.microsoft.com/en-us/download/details.aspx?id=17718

To run WorkedExNet12.exe, WorkedExVbNet12.exe install "Visual C++ Redistributable for Visual Studio 2012 Update 1"
http://www.microsoft.com/en-us/download/details.aspx?id=30679
and "Microsoft .NET Framework 4.5"
http://www.microsoft.com/ru-ru/download/details.aspx?id=30653
